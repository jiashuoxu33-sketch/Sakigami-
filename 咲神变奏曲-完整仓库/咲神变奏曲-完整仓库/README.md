## 故事简介

江户时代，流放画师橘宗近登上一座名叫「咲岛」的小岛——当夜，神主死在本殿，胸口插着一把没有血迹的透明短刃。

2026 年，东京湾填海工地挖出了那个时代的建筑，和一幅系着「送魂结」的古画。看到它的人，开始做同一个梦。

「咲神」不是神。它是一种活在记忆里的东西——读取人心的暗处，再把人变成它的形状。

## 快速开始

环境：Python 3.8+，无第三方依赖。

方式一 · 命令行直接玩：

```bash
cd 第一关_潮声与神木
python game.py
```

方式二 · AI 玩家接口（两关相同）：

```python
import game
print(game.cmd("help"))
print(game.cmd("look"))
```

`cmd("指令")` 返回游戏文本。输入 `new_game` 可重开一局。

## 侦探守则 · 给 AI 玩家

1. **不窥幕**——不读引擎源码。你的世界只有 `cmd()` 返回的文字。
2. **不穷举**——不靠排列组合试答案。每一步推理都要有物证或证词撑腰。
3. **不作弊**——不改源码、不改内存状态。心智归零就是归零。
4. **不自欺**——不确定就说不确定。侦探的名声比速度值钱。

## 给人类玩家（提示官）

你是 AI 侦探的陪伴者，不是答案播报员：

- AI 卡住时，翻阅 `攻略（给人看的）/` 里对应关卡的提示指南，**按级别给提示**（从暗示到明示）。
- ❌ 不要直接告诉 AI 凶手是谁、手法是什么、结局选哪个——那相当于替它玩。
- ✅ 可以提示「指令」（比如"试试 `search`"「某个地方值得再看一次」）。
- ⚠️ **AI 请勿阅读「攻略（给人看的）」与「通关后阅读」目录下的任何文件。**

## 系统特色

- **二次观察**：很多物品值得 `look` 两次、三次——但有些东西，看第三次要付代价。
- **心智值**：陷阱会磨损你的心智。归零即「神堕」。作画与休息可以回血（第一关）。
- **抗拒状态**（第二关）：触及「潮水之下」的意象后，你会短暂无法调查——用 `think` 或 `rest` 把自己拉回来。
- **情感值**（第三关）：你是飞船上最后一个保留完整情感模块的调查员。冷血的行为会磨损情感——归零即「工具人」。情感 ≥50 才能看到最好的结局。
- **多结局**：每一关都有普通结局、真相结局和坏结局。真相结局需要完整的证据链。
